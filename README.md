# gh0st_key
-Is hash cracker program written in python. 

# Requirements
-hashlib

-Termcolor

# Supported Hashes
-blake2b

-blake2s 

-MD4

-MD5

-SHA1 

-SHA224 

-SHA-256 

-SHA-384 

-SHA512 

# CMD
-Write gh0st -h to initiate the program  

-Use command 'info' to see program info
